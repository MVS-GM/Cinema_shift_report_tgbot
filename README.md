Функционал index.html представляет собой App для подсчета стоимости смены с локальным хранением данных и управлением историей. Вот краткое описание:
Вкладки:
  Подсчет: Форма для ввода данных о смене (проект, сотрудник, время, стоимость и др.) с кнопкой "Рассчитать". Рассчитывает стоимость с учетом базовой смены, переработки (прогрессивная шкала: 100% до 8 ч, 200% до 13 ч, 400% с 14 ч), обеда и доп. затрат.
  История подсчетов: Список сохраненных записей с заголовками (Проект_Дата_Сотрудник_Стоимость). Поддерживает раскрытие отчета, удаление (с подтверждением), редактирование и копирование.
  Редактирование: Скрытая вкладка (появляется при нажатии "Изменить"), идентична "Подсчету", но с кнопкой "Сохранить изменения" (желтого цвета #ffbb00) для обновления записи.
Логика:
  Рассчитывает стоимость смены с учетом ночных смен (смена с 16:00 до 03:59 уменьшает базовую длительность на 1 час).
  Сохраняет данные локально в localStorage как массив JSON.
  При редактировании сохраняет исходные startTime, endTime и totalHours, позволяя менять только стоимость и доп. параметры.
Функции:
  Добавление транспорта и оборудования (динамические списки).
  Удаление записей с подтверждением.
  Копирование данных в форму "Подсчет" для создания новой записи.
  Отправка данных боту через Telegram Web App API.
Интерфейс:
  Вкладки переключаются через верхнюю панель.
  История отображает записи вертикально с кнопками действий (желтый "Изменить", зеленый "Копировать", красный "Удалить").
  Всё реализовано на HTML, CSS и JavaScript без серверной части, с хранением данных в браузере пользователя.
